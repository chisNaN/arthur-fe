Why JavaScript became THE actual (2017) standard web language?

  • JavaScript was embedded as the browser client language in 1995. (Still it is the unique one!)
  • Since 2009 a new server backend language, Node.js was built and is using JavaScript syntax
  • Since 2009 also JavaScript is the language base for Google Apps Script
  • Since 2010 when the first commits of PouchDB occurred on Github,
    JavaScript can interact DIRECTLY into NoSQL databases thus WITHOUT needing a server backend language.
  • In 2015 Facebook announced React Native libraries using JSX and JavaScript to build mobile NATIVE apps

Once one knows an ECMAScript standard language
it is quite swift to get into another ES like languages such as ActionScript, JScript etc.
